# Mansplain
Man page PDF viewer featured in Luke Smith's "Layin' down Pipes like a Unix Chad" video
## Dependencies
* awk
* dmenu
* man
* zathura
* zathura-pdf-mupdf

# Installation
```
git clone https://github.com/Wizkid116/mansplain && cd ~/mansplain
chmod +x mansplain

#optional
sudo mv mansplain /bin
```
